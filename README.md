# Start project

---> Compile sass -> css
1. npm init
2. npm install node-sass --save-dev
3. npm install concat --save-dev
4. npm install autoprefix --save-dev
5. npm install postcss-cli -save-dev
6. npm install npm-run-all --save-dev
7. npm run build:css

---> Run live server
1. npm install live-server
2. npm run start
